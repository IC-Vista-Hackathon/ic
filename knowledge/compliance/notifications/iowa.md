# Iowa — Notification Law Reference

> **Internal compliance reference only — not legal advice.** Verify all citations with counsel before relying on them for a specific incident or contract decision.

## 1. Data Breach Notification

**Statute:** Personal Information Security Breach Protection, Iowa Code Chapter 715C.

- **Trigger:** Unauthorized acquisition of personal information maintained in computerized form (or paper form transferred from computerized form) that compromises its security, confidentiality, or integrity. Iowa Code 715C.1, 715C.2.
- **Consumer notice deadline:** No fixed day count — notice must be given "in the most expeditious manner possible and without unreasonable delay," subject to law-enforcement delay requests. Iowa Code 715C.2(1).
- **State AG notice:** Required if the breach affects **more than 500 Iowa residents**; must be given **within 5 business days after** notice is given to consumers. Iowa Code 715C.2(2).
- **Consumer reporting agencies:** No separate mandatory direct-notice trigger identified; consumer notices must include CRA contact information as content, per Iowa AG guidance.
- **Payment card number + security code as "personal information":** **Yes.** Definition includes a financial account, credit, or debit card number in combination with any required security code, access code, or password permitting account access. Iowa Code 715C.1.
- **Encryption safe harbor:** **Yes.** "Personal information" only includes data elements that are unencrypted/unredacted, or encrypted/redacted data where the decryption/unredaction keys were also obtained in the breach. Iowa Code 715C.1.
- **Penalties:** A violation is an unlawful practice under Iowa Code 714.16; Attorney General may seek damages on behalf of injured consumers. Iowa Code 715C.2(4). No fixed per-violation civil penalty amount was identified in the sections reviewed — **not confirmed, verify with counsel.**
- **Private right of action:** Not confirmed — no explicit private right of action found; enforcement runs through the Attorney General. Statute notes remedies are cumulative with other law (e.g., Iowa Code 714.16B identity-theft civil actions), which may be relevant but is a distinct cause of action.

## 2. Consumer-Facing Payment / Billing Notices

- **No general automatic-renewal/subscription notice statute identified for Iowa.** Iowa has narrow, sector-specific cancellation-notice rules instead:
  - Residential/motor vehicle service contracts: if the *company* cancels, written termination notice required at least 15 days before the termination date; contract must clearly state cancellation terms. Iowa Code Chapter 523C (523C.7).
  - Physical exercise club contracts: separate right-of-cancellation regime. Iowa Code Chapter 552.
  - General home-solicitation sales: 3-business-day right to cancel transactions ≥ $25 made at home/workplace for personal/family/household goods or services. Iowa Code Chapter 555A (cited via Iowa AG "Right to Cancel" guidance).
- **Price-increase notice, autopay/subscription renewal notice, cancellation-confirmation notice generally:** Not confirmed — no general Iowa statute located requiring these outside the narrow sectors above. Verify with counsel.

## 3. State-Specific EFT/ACH Notice Requirements

- Iowa Code Chapter 527 (Electronic Transfer of Funds) exists but appears to largely track/incorporate the federal EFTA/Reg E framework rather than adding distinct consumer notice timing requirements.
- No Iowa-specific consumer notice requirement beyond federal Reg E (preauthorized transfers, change-in-terms) was confirmed. **Not confirmed — verify with counsel.**
